# Smart Traffic System 🚦

**Smart Traffic System** is a Python-based project for simulating and optimizing traffic light control using machine learning. It analyzes real-time traffic density, waiting times, and emergency vehicles to improve traffic flow.

---

## Project Overview

- Real-time simulation of multiple traffic lanes.
- Predictive ML model to optimize green light duration based on lane congestion.
- Emergency vehicle prioritization for immediate green lights.
- Calculates lane waiting times and traffic density dynamically.
- Supports CSV input data for traffic analysis.

---

## Features

1. **Traffic Density Simulation** – Tracks lane congestion using random or historical data.
2. **Machine Learning Optimization** – Uses RandomForestClassifier for traffic light control.
3. **Emergency Vehicle Handling** – Prioritizes lanes with emergency vehicles.
